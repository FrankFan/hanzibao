# 汉字学习 App — 设计风格指南

## 整体风格定位

**暖系儿童教育风**。以橙色为主调，米白色为底，整体传递出温暖、轻松、亲切的氛围，契合小学生群体的审美与认知习惯。视觉上克制而不花哨——没有多余的装饰，用颜色和圆角建立层次，让汉字本身成为页面的主角。

## 圆角系统

整体采用**大圆角**策略，圆润感传递友好与安全感，适合儿童产品。

| 元素 | 圆角值 |
|------|--------|
| Logo 图标 | `rounded-3xl`（24px，接近圆形方块） |
| 田字格容器 | `rounded-2xl`（16px） |
| 卡片（释义、词组） | `rounded-2xl`（16px） |
| 常用字方块 | `rounded-xl`（12px） |
| 词组标签 chip | `rounded-xl`（12px） |
| 控制按钮 | `rounded-full`（完整圆形） |
| 速度选择按钮 | `rounded-full`（胶囊形） |
| 加载骨架屏 | `rounded-full` |

## 字体系统

```
font-family: -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif
```

- 优先使用系统字体，避免引入中文字体包（体积过大）
- PingFang SC（苹果）、Microsoft YaHei（Windows）覆盖主流中文环境
- 汉字展示（搜索框、常用字方块）额外追加 `serif` 衬线字体，增强书法感

### 字号层级

| 场景 | 字号 | 字重 |
|------|------|------|
| 搜索框输入字 | `text-4xl`（36px） | `font-bold` |

## 间距与布局

- 页面内容最大宽度：`max-w-xs`（320px）/ `max-w-sm`（384px），窄列居中，移动端友好
- 页面垂直方向用 `gap-6`（24px）分隔各功能区块
- 卡片内边距：`p-4`（16px）
- 常用字格栅：`grid-cols-6`，`gap-3`（12px），等比正方形

## 交互与动效

### 按钮反馈
所有可点击元素统一使用 `active:scale-95`，按下时轻微缩小，给儿童清晰的点击感知。过渡时间 `duration-100 ~ 150ms`，快速响应。

### 悬停态
- 卡片按钮：边框颜色加深（`orange-200 → orange-400`），背景变浅橙（`orange-50`）
- 常用字方块：边框加深 + `shadow-md`，轻微抬起感
- 发音按钮：背景加深（`orange-100 → orange-200`）

### 加载态
使用 `animate-pulse` 骨架屏占位，形状与实际内容一致（拼音区：圆角长条；释义区：多行不等宽圆角条），避免布局跳动。

### 田字格动画
Hanzi Writer SVG 逐笔描绘，笔画颜色 `#1c1917`（近黑），轮廓色 `#d1d5db`（浅灰），绘制色 `#f97316`（主橙色）。页面加载后自动播放，给孩子即时的视觉反馈。

## 组件速查

### Logo
橙色圆角方块（`rounded-3xl`）+ 白色衬线「字」，配页面标题和说明副标题，三行垂直居中。

### 搜索框
无边框、透明背景、超大字号，底部一条细橙线作为焦点指示，极简到只剩输入体验。

### 田字格
米白底色 + 浅灰中轴线（`::before` / `::after` 伪元素），外框 `rounded-2xl` 圆角，整体有轻微阴影（`shadow-sm`）与页面底色区分。

### 控制区
三个圆形按钮水平排列（重播、播放/暂停、显示全字），中间主按钮更大（`w-16 h-16`），两侧辅助按钮标准尺寸（`w-12 h-12`）。速度选择为一排胶囊按钮，激活态填充橙色实底。

### 释义卡片
白色圆角卡片，标题为小型橙色大写标签，内容按序号列表展示。词组为独立的 chip 网格，每个 chip 竖排显示汉字 + 拼音 + 释义。

---

## 设计原则总结

1. **儿童优先**：大字号、大圆角、高对比度橙色，降低认知负担
2. **汉字为主角**：页面留白充足，田字格居中突出，其他元素退为配角
3. **暖而不燥**：橙色作为强调色，大面积使用米白和白色中和视觉刺激
4. **即时反馈**：每个交互都有明确的视觉响应（缩放、颜色变化、动画）
5. **移动端优先**：窄列布局、触控友好的点击区域（最小 `w-9 h-9`，36px）
